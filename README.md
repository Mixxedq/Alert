# 📡 Trading Alerts Bot

Telegram-бот для мониторинга аномальных движений и консолидаций на криптовалютных активах.

## 🚀 Быстрый старт

### 1. Создайте бота в Telegram
1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте токен вида `110201543:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw`

### 2. Установите зависимости
```bash
pip install aiohttp
```

### 3. Настройте токен
```bash
cp .env.example .env
# Отредактируйте .env и вставьте ваш BOT_TOKEN
```

### 4. Запустите бота
```bash
python bot/bot.py
```

## 📊 Типы алертов

### ⚡ Аномальное движение (SPIKE)
Срабатывает когда последняя **закрытая** свеча переместилась более чем на N% от открытия до закрытия.

- По умолчанию: **3%** за одну свечу
- Настраивается через `/set SYMBOL spike N`

### 🔷 Консолидация
Срабатывает когда **N последних баров** все имеют диапазон (high−low) меньше 30% от ATR(14).

- По умолчанию: **10 баров**
- Настраивается через `/set SYMBOL bars N`
- Сигнал о возможном сильном выходе из диапазона!

## 🤖 Команды бота

| Команда | Описание |
|---------|----------|
| `/watch BTCUSDT 1h` | Добавить актив (символ + таймфрейм) |
| `/unwatch BTCUSDT` | Удалить актив |
| `/list` | Список всех отслеживаемых активов |
| `/set BTCUSDT spike 5` | Порог спайка: 5% |
| `/set BTCUSDT bars 15` | Консолидация: 15 баров |
| `/set BTCUSDT cooldown 30` | Кулдаун алерта: 30 мин |
| `/pause` | Приостановить алерты |
| `/resume` | Возобновить алерты |
| `/status` | Статус и количество активов |
| `/help` | Справка |

## 💹 Поддерживаемые активы

Бот использует **Binance API** (бесплатно, без API-ключа для публичных данных).

Примеры символов:
- `BTCUSDT`, `ETHUSDT`, `SOLUSDT`
- `BNBUSDT`, `XRPUSDT`, `DOGEUSDT`
- Любой спотовый или фьючерсный символ с Binance

## 🔧 Параметры (config)

```python
DEFAULT_SPIKE_THRESHOLD = 3.0        # % движение свечи → спайк алерт
DEFAULT_CONSOLIDATION_BARS = 10      # баров подряд с низкой волатильностью
DEFAULT_CONSOLIDATION_ATR_MULT = 0.3 # диапазон < 30% ATR(14)
DEFAULT_CHECK_INTERVAL = 60          # секунд между проверками
```
