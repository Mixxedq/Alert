ЗАПУСК БОТА
1. Установите зависимости:
pip install aiohttp

2. Задайте токен бота в файле .env:
BOT_TOKEN=ваш_токен_здесь

3. Запустите бота:
python bot.py
КОМАНДЫ TELEGRAM БОТА
/watch BTCUSDT 1h

/status

КАК РАБОТАЮТ АЛЕРТЫ
⚡ АНОМАЛЬНОЕ ДВИЖЕНИЕ (SPIKE)
Срабатывает когда последняя закрытая свеча переместилась более чем на N% (по умолчанию 3%) от открытия до закрытия.

Учитывается только тело свечи (open → close).
🔷 КОНСОЛИДАЦИЯ
Срабатывает когда N последних баров (по умолчанию 10) имеют диапазон high–low меньше 30% от ATR(14).

Сигнал о возможном сильном выходе из диапазона.
